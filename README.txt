# SocialBlock Economy Mod

Un mod Minecraft Forge 1.20.1 ajoutant un système économique basé sur les interactions sociales.

## Fonctionnalités principales
- [ ] Système de monnaie virtuelle
- [ ] Blocs interactifs sociaux
- [ ] Interface de trading

## Pour commencer

### Prérequis
- Java JDK 17
- Minecraft 1.20.1
- Forge MDK
